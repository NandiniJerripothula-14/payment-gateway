# Videos Directory

This directory contains video demonstrations of the Payment Gateway application.

## Video Files

### Complete Demo Video
- `payment-gateway-demo.mp4` - Full end-to-end demonstration (3 minutes)
  - System startup with Docker
  - Dashboard overview and credentials
  - API order creation
  - UPI payment flow
  - Card payment flow
  - Success and failure states
  - Database verification

## How to Add Video

### Option 1: Upload to YouTube (Recommended)
1. Visit https://www.youtube.com/upload
2. Upload your recorded video
3. Set visibility to "Unlisted"
4. Update the main README.md with the YouTube link

### Option 2: Store Video Locally
1. Place your video file (MP4 format) in this directory
2. Name it `payment-gateway-demo.mp4`
3. Link to it from README.md as: `videos/payment-gateway-demo.mp4`

### Video Recording Guide

See [VIDEO_DEMO_SCRIPT.md](../VIDEO_DEMO_SCRIPT.md) in the root directory for complete instructions on recording a professional demo video.

## Technical Details

**Recommended Specifications:**
- Format: MP4 (H.264)
- Resolution: 1920x1080 or 1366x768
- FPS: 30 fps
- Bitrate: 5-8 Mbps
- Duration: 2-3 minutes

**File Size:**
- Typically 200-400 MB for a 3-minute demo
- If > 100 MB, consider storing on YouTube instead
